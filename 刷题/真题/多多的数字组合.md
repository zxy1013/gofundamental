多多君最近在研究某种数字组合：定义为：每个数字的十进制表示中(0~9)，每个数位各不相同且各位数之和等于N，满足条件的数字可能很多，找到其中的最小值即可。多多君还有很多研究课题，于是多多君找到了你--未来的计算机科学家寻求帮助。 

```python
x = int(input())
if x < 10:
    print(x)
# 因为1+2+3+4+5+6+7+8+9等于45,大于45就不能凑了
elif x > 45:
    print('-1')
else:
    # 现在表示的是哪一位
    count = 0
    re = 0
    for i in range(9,-1,-1):
        # 低位分配大数
        if i <= x:
            re += i*pow(10, count)
            count += 1
            x -= i
        if x == 0:
            print(re)
            break
```
